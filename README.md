# 🔎 Job Scraper Automation Tool

Python automation tool that scrapes job listings, filters results by keywords, and stores structured data in CSV format while avoiding duplicates.

## 🚀 Features
- Web scraping with BeautifulSoup  
- Keyword-based filtering  
- CSV storage with deduplication  
- Logging system  
- Modular architecture
